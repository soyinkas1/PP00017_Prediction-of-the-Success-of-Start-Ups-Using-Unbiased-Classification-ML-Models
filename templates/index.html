Introduction

This project, originally undertaken as dissertation research work for the award of M.Sc Data Science from Cardiff Metropolitan University, has received an A grade.

The current version of the project aims to enhance the approach and experiments conducted during the initial research phase. Additionally, it will implement additional best practices to further improve the project's outcomes.

Key Objectives

  • Improve upon the experiments conducted.
  • Implement additional best practices in project implementation.

The project will focus on refining existing methodologies, exploring new techniques, and incorporating the latest advancements in data science. This will involve rigorous analysis, experimentation, and validation to ensure the project's success.

Expected Outcomes

By enhancing the project's approach and experiments, as well as implementing additional best practices, we anticipate achieving superior results compared to the initial research phase. The refined project aims to contribute significantly to the field of data science.
